Мои проекты

  • rep_searcher

    Проект на react и ts, производит поиск по репозиториям на Github с помощью GitHub REST API и RTK Query.

  • prey_API

    Проект, посвященный игре Prey на express, содержит в себе сервер, реализующий GET-запросы по получению информации из базы данных Mongodb.

  • prey_API_client

    Сайт-клиент в дополнение к prey_API на react, содержит в себе функционал ввода запроса пользователем, вывода ответа сервера и документацию.

  • react_star_wars

    Сайт на react, который отправляет GET-запросы к SWAPI, содержит в себе функционал вывода карточек персонажей Star Wars и информации о каждом персонаже, функционал поиска по имени и добавления персонажа в избранное.

  • to_do_list

    Приложение списка дел на js, реализует запись задачи, ее удаление и выполнение (чек-бокс), запись задачи может производиться на любой день из встроенного календаря, данные хранятся в localStorage.

  • constructor_labels

    Приложение конструктора лэйблов на js, реализует разные модификации лэйбла и вывод css-кода для последующего копирования.
